# Security Policy

## Supported Versions

Currently, we only provide security updates for the latest stable version of FinTrack.

| Version | Supported          |
| ------- | ------------------ |
| 0.x.x   | :white_check_mark: |
| < 0.1   | :x:                |

## Reporting a Vulnerability

We take the security of FinTrack seriously. If you discover a security vulnerability, please do not open a public issue. Instead, please report it via one of the following methods:

1.  **GitHub Private Reporting:** Use the [Private Vulnerability Reporting](https://github.com/BODMAT/FinTrack/security/advisories/new) feature on GitHub.
2.  **Email:** Contact the maintainers directly at `fintrack.community@gmail.com`

Please include as much detail as possible, including steps to reproduce the vulnerability and a proof-of-concept if available.

### Our Response Process

- We will acknowledge your report within 48 hours.
- We will investigate and provide a timeline for a fix.
- Once fixed, we will coordinate a public disclosure and publicly acknowledge your contribution.
